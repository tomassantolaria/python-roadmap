# Python Automation Projects

Este directorio contiene proyectos prácticos enfocados en automatización con Python, orientados al desarrollo de habilidades aplicables a entornos laborales reales.

## Objetivos
- Automatización de tareas repetitivas
- Procesamiento de archivos
- Limpieza de datos
- Integración con APIs
- Desarrollo progresivo de portfolio profesional

---

## Proyectos incluidos

### 1. File Processor
Script que:

- Lee archivos de texto
- Limpia palabras eliminando puntuación
- Excluye valores numéricos
- Cuenta palabras válidas

### Tecnologías utilizadas:
- Python
- pathlib
- Manejo de archivos
- Manejo de errores

### Habilidades desarrolladas:
- Modularización
- Limpieza de datos
- Buenas prácticas iniciales
- Estructuración de proyectos
